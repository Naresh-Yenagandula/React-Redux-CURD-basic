# React CURD app with Redux & Json-Sever

### Technology stack

1. React Js
2. Redux
3. Json-Server

### Set Up

1. Clone repository
2. Open terminal and execute
   *** npm install ***
3. Then do
   *** npm start ***
4. Open another terminal and execute cmd to start Json-Server
   *** npm run server ***
5. Open to browser and go to
   *** http://localhost:3000 ***
