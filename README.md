# React CURD app with Redux & Json-Sever

### Technology stack

1. React Js
2. Redux
3. Json-Server

### Set Up

1. Clone repository
2. Open terminal and execute
   **_ npm install _**
3. Then do
   **_ npm start _**
4. Open another terminal and execute below cmd to start Json-Server
   **_ npm run server _**
5. Open to browser and go to
   **_ http://localhost:3000 _**
